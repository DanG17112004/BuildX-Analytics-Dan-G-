# 🎓 College Placement Data Analysis

## 📌 Project Overview

This project was completed as part of the **BuildX Data Analytics Bootcamp Bonus Project**. It demonstrates an end-to-end data analytics workflow using Python, SQL, Power BI, and GitHub.

---

## 📂 Dataset

**Dataset:** College Placement Records

### Why I Selected This Dataset

I selected this dataset because it provides valuable insights into student placements, salaries, academic performance, and specialization trends. It is ideal for learning business analytics and dashboard development.

---

## 🛠 Tools Used

- Python (Pandas)
- SQL
- Power BI
- GitHub

---

## 📊 Business Questions Answered

1. How many students got placed?
2. Which specialization has the highest average salary?
3. What is the overall placement rate?

---

## 📈 Key Insights

- Total Students: **215**
- Placed Students: **148**
- Placement Rate: **68.8%**
- Average Salary: **₹289K**
- Marketing & Finance specialization receives the highest average salary.
